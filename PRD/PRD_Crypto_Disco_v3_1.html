1. Visi & Ekonomi Ekosistem (Executive Summary)

Crypto Disco adalah platform gamifikasi Web3 di jaringan Base yang menghubungkan aktivitas sosial (Farcaster/Twitter) dengan ekonomi on-chain. Versi 3.2 memperkenalkan UGC Raffle sebagai pendorong utama revenue platform, di mana 20% biaya platform didistribusikan kembali ke pemegang SBT (Soulbound Token) berdasarkan tier (Diamond hingga Bronze).

3. Arsitektur & Global Workflow

3.1 Technical Architecture

  • Frontend: React (Vite) + Lucide + Wagmi (Base Sepolia/Mainnet).
  • API Bundles: Vercel Serverless (user-bundle, tasks-bundle, raffle-bundle).
  • Database: Supabase (PostgreSQL) dengan RLS & Trigger otomatis.
  • Contracts: DailyAppV13, RaffleV2, MasterX Revenue Share.

5. Database Architecture & Logic

5.1 Multi-Layer XP Sync

Sistem menggunakan Database Triggers (trg_sync_xp_on_claim) untuk memastikan total_xp selalu akurat berdasarkan riwayat klaim di tabel user_task_claims. Hal ini mencegah manipulasi XP manual.

5.2 Dynamic Tier Ranking (Percent-Rank)

View v_user_full_profile menggunakan fungsi window PERCENT_RANK() untuk menentukan tier user secara real-time:

  • Diamond: Top 1% Global XP
  • Platinum: Top 5% Global XP
  • Gold: Top 15% Global XP
  • Silver: Top 40% Global XP
  • Bronze: Top 80% Global XP

6. Admin Control Hub (Lurah Dashboard)

Terletak di /admin, pusat kendali ini memungkinkan:

  • Economy Tuning: Mengubah biaya platform & multiplier XP secara live.
  • Infrastructure Management: Mengalihkan alamat kontrak (DailyApp, USDC, Raffle) tanpa redeploy frontend.
  • Role Control: Mengatur siapa yang bisa menjadi Verifier Task.
  • Raffle Ops: Memantau partisipasi tiket dan distribusi prize pool.

🛡️ 7. Security & Zero-Leak Protocol

Aplikasi ini tunduk pada Ecosystem Sentinel Mandate:

  • Gitleaks Check: Pre-push hook wajib untuk mencegah kebocoran secret.
  • Security Invoker: Seluruh Query DB frontend menggunakan RLS yang ketat.
  • Signature Verification: Seluruh endpoint API sensitif memerlukan EIP-191 signature.
